
# Silicon Commodity Risk Dashboard

✅ Fully functional  
✅ No API keys required  
✅ Runs offline with synthetic data  

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

This project demonstrates:
- Silicon price candlestick (10Y)
- VIX, DXY, Gold, Silver, Copper integration
- Correlation analysis
- Simple predictive trend logic
